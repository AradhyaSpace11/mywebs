HTML Elements Explained

This document demonstrates various HTML elements with explanations and examples to help you understand their usage from basic to advanced levels.

1. Text Elements

Headings (h1 to h6)

The <h> tags are used for headings, with <h1> being the highest level and <h6> the lowest.

Paragraphs (p)

The <p> element represents a paragraph of text.

This is an example of a paragraph in HTML. Paragraphs automatically add spacing around the text, making it easier to read.

Links (a)

The <a> element is used to create hyperlinks, which allow users to click and navigate to another page or location.

2. Division (div)

The <div> tag is a block-level element used to group content. It can be styled with CSS to control layout.

This content is inside a div element with a border.

3. Buttons (button)

The <button> element creates a clickable button. It can trigger JavaScript functions.

4. Input Fields (input)

Input fields are created using the <input> tag. You can specify different types like text, password, email, etc.

5. Images (img)

The <img> tag is used to display images. Use the src attribute to specify the image URL.

6. Media Elements: Audio and Video

Audio (audio)

The <audio> tag is used to embed audio files. Use the controls attribute to show playback controls.

Video (video)

The <video> tag embeds video files. The controls attribute displays playback controls.

7. Lists

Ordered List (ol)

Use <ol> for ordered lists, where each item is numbered.

  1. First item
  2. Second item
  3. Third item

Unordered List (ul)

Use <ul> for unordered lists with bullet points.

  • Item one
  • Item two
  • Item three

8. Tables

The <table> element is used to display data in a tabular format. It contains rows (<tr>), header cells (<th>), and data cells (<td>).

Name Age
Alice 30
Bob 25

9. Inline Frame (iframe)

The <iframe> tag is used to embed another HTML document within the current page.

10. Forms

The <form> element is used to collect user input. It can include various input types and allows data submission.
